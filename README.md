# AI Network Security Lab

This project extends an academic OSPF lab into an AI-assisted cybersecurity and network observability platform.

It combines:
- OSPFv2 / OSPFv3 routing
- VLAN-based virtual topology
- monitoring with Grafana/Prometheus/Loki
- IDS analysis with Suricata or Zeek
- local LLM inference on Apple Silicon
- RAG over networking and cybersecurity documentation
